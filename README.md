# ai-interview-platform
AI 面试题生成平台
根据输入生成面试题，有收藏，历史记录等功能
如何使用：
1.打开spring 填写spring src/main/resources/application.yml 补全数据库信息与smtp
2.打开rag pip install -r requirements.txt
3.导入database.sql
4.打开nginx 启动nginx.exe
5.依次启动redis，spring，rag/main.py
6.浏览器访问localhost
